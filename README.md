# HTML → Poster PDF

Convert any HTML file to a print-ready poster PDF at any size.

🌐 **Live:** https://html-to-poster-production.up.railway.app

## Features
- Drag & drop any `.html` file
- Preset sizes: 48×36 in, 36×48 in, A0, A1, and more
- Custom width/height in inches, cm, mm, or px
- Downloads a print-ready PDF instantly

## Run Locally

```bash
npm install
npm start
# Open http://localhost:3000
```

## Deploy

Deployed via [Railway](https://railway.app) using Docker. Chromium is installed automatically via the Dockerfile.
